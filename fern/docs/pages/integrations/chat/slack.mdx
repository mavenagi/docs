## Setup within Maven
Each Agent that should be visible within Slack must be enabled and given a name within the Slack settings page.
If the "Enable this agent for Slack" checkbox is not checked, the agent will not be available within Slack.

## Setup within Slack
Once the settings are configured, Maven needs to be installed into your Slack workspace.
On the settings page for the Slack integration within the Maven dashboard, click the link to add the 
Maven Slack app to your workspace. This will take you through an authentication flow with Slack.

You should then have a Slack app which can be chatted directly or added to a channel.
