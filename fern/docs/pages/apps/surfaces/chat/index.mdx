Maven offers a chat widget that allows you to embed a Maven experience directly into your website.

## Quick Start

<Steps>
  <Step title="Install the chat app">
    - Navigate to the **Apps** page in the **Agent Designer**
    - Click on **Browse & Install** on the **Apps Directory** tile
    - Search for "Chat" and click on the **Chat** app tile
    - Click on the **Install** button
  </Step>
  <Step title="Add to your website">
    Your personalized code snippet can be found on the chat settings page under the "Instructions" tab. Copy and paste the following code snippet into your website's HTML, preferably just before the closing `</body>` tag:
    
    ```html
    <script src='https://chat.onmaven.app/js/widget.js' defer></script>
    <script async>
      addEventListener("load", function () {
        Maven.ChatWidget.load({
          organizationId: "YOUR_ORGANIZATION_ID",
          agentId: "YOUR_AGENT_ID",
        })
      });
    </script>
    ```
  </Step>
</Steps>

<Note>
In order for the chat widget to load upon installation, you must first update the Content Security Protocol on your site to allow content from `chat.onmaven.app`. If your website uses a Content Security Policy (CSP), make sure to add `chat.onmaven.app` to your allowed domains for script-src, connect-src, and frame-src directives to enable proper functionality of the chat widget.
</Note>

## Live Agent Handoff

Connect your chat widget to live agent support through various service providers. When configured, conversations can be automatically transferred to human agents.

Currently supported providers:
- [Salesforce (Legacy)](/apps/surfaces/chat/live-agent-handoff/salesforce-legacy)
- [Salesforce Messaging](/apps/surfaces/chat/live-agent-handoff/salesforce-messaging)
- [Zendesk](/apps/surfaces/chat/live-agent-handoff/zendesk)
- [Front](/apps/surfaces/chat/live-agent-handoff/front)
- [HubSpot](/apps/surfaces/chat/live-agent-handoff/hubspot)
- [Genesys](/apps/surfaces/chat/live-agent-handoff/genesys)

See the [Live Agent Handoff Overview](/apps/surfaces/chat/live-agent-handoff) for more information.
