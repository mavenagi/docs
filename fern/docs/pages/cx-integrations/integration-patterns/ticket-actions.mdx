title: Ticket Actions
description: Create, update, and manage tickets programmatically
---

Ticket Actions enable Maven to create and manage tickets in your helpdesk system as part of automated workflows.

## What It Does

- **Ticket Creation**: Automatically create tickets from chat conversations
- **Status Updates**: Update ticket status, priority, and fields
- **Assignment**: Route tickets to appropriate agents or queues
- **Tagging**: Apply tags and categories for organization

## Supported Platforms

Ticket Actions are available for:

- [Zendesk](/docs/cx-integrations/zendesk)
- [Salesforce](/docs/cx-integrations/salesforce)
- [Freshdesk](/docs/cx-integrations/freshdesk)
- [HubSpot](/docs/cx-integrations/hubspot)
- [Intercom](/docs/cx-integrations/intercom)

## Common Use Cases

- Create a ticket when Maven can't fully resolve an issue
- Escalate urgent issues with appropriate priority
- Log all AI conversations as tickets for record-keeping
- Update existing tickets based on follow-up conversations

See your platform-specific guide for configuration options.
