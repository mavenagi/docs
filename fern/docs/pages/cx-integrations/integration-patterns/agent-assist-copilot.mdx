The Agent Assist Copilot is an AI-powered sidebar that integrates directly into your helpdesk interface, providing real-time assistance to customer service agents.

## What It Does

- **Smart Suggestions**: AI-generated response recommendations based on conversation context
- **Knowledge Lookup**: Instant access to relevant knowledge base articles
- **Response Drafting**: Generate draft responses that agents can review and send
- **Conversation Summary**: Quick summaries of long ticket threads
- **Action Recommendations**: Suggested next steps based on customer intent

## Supported Platforms

The Agent Assist Copilot is available for:

- [Zendesk](/docs/cx-integrations/zendesk)
- [Salesforce](/docs/cx-integrations/salesforce)
- [Freshdesk](/docs/cx-integrations/freshdesk)
- [HubSpot](/docs/cx-integrations/hubspot)
- [Front](/docs/cx-integrations/front)
- [Genesys](/docs/cx-integrations/genesys)
- [HelpScout](/docs/cx-integrations/helpscout)

## How It Works

The copilot appears as a sidebar panel within your existing helpdesk interface. As agents work through tickets, Maven analyzes the conversation in real-time and provides contextual assistance.

See your platform-specific guide for installation and configuration instructions.
