Maven integrates with leading customer service and helpdesk systems, enabling AI-powered support across your existing CX stack.

## Integration Patterns

Maven offers several integration patterns depending on your CX system and use case:

| Pattern | Description |
|------------|-------------|
| **Agent Assist Copilot** | AI sidebar that helps agents with real-time suggestions, knowledge lookup, and response drafting |
| **Live Agent Handoff** | Seamlessly transfer chat conversations from Maven to human agents |
| **Conversation Sync** | Sync ticket and conversation data for AI training and analytics |
| **Ticket Actions** | Create, update, and manage tickets programmatically |
| **Knowledge Connectors** | Import knowledge base content from your CX system |
